# UTS Sistem Perpustakaan

Project Java OOP menggunakan Strategy Pattern dan Factory Pattern.

## Fitur
- Input nama anggota
- Pilih jenis anggota (Mahasiswa, Dosen, Umum)
- Menampilkan daftar buku
- Proses peminjaman
- Menghitung keterlambatan
- Menghitung denda otomatis

## Konsep
- OOP
- SRP
- OCP
- Strategy Pattern
- Factory Pattern
